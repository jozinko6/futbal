# REFERENCE STYLE — Arkádový futbal zo začiatku 90. rokov

## Referencia (všeobecná inšpirácia, nie kópia)
- Typ kamery: diagonálny pohľad z tribúny, horizontálne scrolluje
- Tempo: rýchle, okamžitá odozva
- Veľkosť postáv: veľké, detailné (40-56px)
- Ovládanie: joystick + 3 tlačidlá (A/B/C)
- Farebná kompozícia: sýta zelená tráva, jasné biele čiary, pestrofarebné bannery
- HUD: skóre, čas, tímové skratky, 1P/2P, Special Units, Captain Meter
- Výber kapitána: pred zápasom z kandidátov
- Špeciálne strely: Super Shot, Hyper Shot
- Turnajový režim: séria zápasov, boss tím
- Špeciálne menu počas prerušenia

## Čo sa NEkopíruje
- Názov Taito Cup Finals
- Logo Taito
- Pôvodné sprity, animácie, portréty
- Pôvodné zvuky, hudba, fonty
- Presné názvy tímov
- ROM dáta, emulovaný kód
